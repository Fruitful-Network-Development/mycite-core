## Current Contract

Sandbox is the shared lifecycle engine for resource staging, compile, decode, adapt, save, and publish flows. Resource inventory ownership remains separate and index-backed.

This topic isolates lifecycle rules from inventory and inherited-context rules so those concerns can evolve without forcing a wide documentation audit.
