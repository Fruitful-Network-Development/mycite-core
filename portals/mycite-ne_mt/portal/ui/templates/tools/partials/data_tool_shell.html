Anthology Table
Layered view with grouped append workflow

Use this as the primary development interface for anthology datums. Expand/collapse by layer and value group. Use + Append datum at the bottom of each value group.

Rule: value_group=0 stores selection references inside the row magnitude payload. value_group>=1 supports multi-pair rows.
