    Coordinate Notation
    Fixed-Width Signed Hex Split

    AGRO ERP decodes each coordinate token by splitting it into equal halves. Upper half is longitude-axis value, lower half is latitude-axis value.

    Each axis is decoded as signed two's-complement and scaled by 1e7 to produce decimal longitude/latitude pairs.
