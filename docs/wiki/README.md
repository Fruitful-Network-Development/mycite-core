# Wiki

This directory is reserved for exploratory notes and future secondary documentation.

- Use [../governance/README.md](../governance/README.md) for the promotion and lifecycle rules that govern wiki content.
- Use [../personal_notes/README.md](../personal_notes/README.md) for preserved personal notes, operational inventories, and archived discussions that should not be treated as shared wiki material.
- It is not authoritative for v2 architecture.
- `docs/wiki/legacy/` is legacy explanatory material, not current modular-v2 authority.
- It must not introduce competing terminology.
- Any rule that matters must be promoted into `docs/ontology/`, `docs/decisions/`, `docs/plans/`, `docs/contracts/`, or `docs/testing/`.

See [../ontology/non_authoritative_zones.md](../ontology/non_authoritative_zones.md).
